# Quiply

A social media application for quick comments and fun sentences.

## Description

* Languages

- HTML
- Tailwind (CSS)
- Javascript


## Getting Started

### Installing

* Clone repo
```
gh repo clone DanielEkerhovd/Quiply
```

* Install dependencies

```
npm install
```

* Run program
```
npm run dev
```
